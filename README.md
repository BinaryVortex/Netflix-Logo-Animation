# Netflix Logo Animation

A small HTML/CSS/JavaScript animation that recreates the Netflix logo entrance effect.

![Animation screenshot](./Screenshot%202024-05-23%20220406.png)

## Overview

This repository contains a lightweight demo that animates the Netflix wordmark (N E T F L I X) using HTML, CSS and a bit of JavaScript. It's intended as a fun exercise in CSS animations and timing.

## Demo / Screenshot

The screenshot above shows the animation in the browser. To view the animation live, open `index.html` in any modern browser.

## Built with

- HTML
- CSS (animations & transforms)
- Minimal JavaScript (to trigger / control the animation)

## How to run

1. Clone or download the repository.
2. Open `index.html` in your browser (double-click or right-click -> Open with -> your browser).

Optional: serve the folder with a simple static server (npm, Python, VS Code Live Server) if you prefer:

- Python 3: `python -m http.server 8000` then visit `http://localhost:8000`
- Node (http-server): `npx http-server .`

## File structure

- `index.html` — markup and animation trigger
- `style.css` — styles and keyframe animations
- `Screenshot 2024-05-23 220406.png` — screenshot used in this README

## Customize

- Edit `style.css` to change colors, timing, or add easing.
- Tweak the JavaScript (if present) to change when the animation starts or to add replay controls.
